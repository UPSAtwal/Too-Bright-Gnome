# Too Bright Gnome

A Simple Gnome extension to control external monitor brightness using ddc/ci.

## Currently tested Gnome Versions

- Gnome 42.5 on Ubuntu 22.04 LTS

*Open a PR and edit the README.md if it works on your device.*
